# Finance App

Aplikasi web sederhana untuk mengelola keuangan pribadi atau bisnis kecil.  
Pengguna dapat mencatat pendapatan, pengeluaran, serta melihat dashboard visualisasi keuangan.

## Fitur

- Autentikasi pengguna (login/logout)  
- Tambah, edit, hapus, dan lihat transaksi  
- Dashboard dengan saldo harian, bulanan, dan tahunan  
- Grafik pie untuk kategori pendapatan dan pengeluaran  
- Tampilan modern dan responsif menggunakan Tailwind CSS  

## Teknologi

- PHP 8+  
- MySQL / MariaDB  
- Tailwind CSS  
- Chart.js  

## Cara Menjalankan

1. Clone repository ini:

git clone https://github.com/username/finance-app.git

    Buat database:

    Nama database: finance_db

    Buat tabel: users, accounts, categories, transactions

    Sesuaikan konfigurasi database di config/db.php

    Jalankan di server lokal (XAMPP, LAMP, MAMP, atau Ubuntu server)

    Buka browser: http://localhost/finance-app/
